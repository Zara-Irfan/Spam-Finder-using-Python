## How to Export Your Gmail Mail Using Google Takeout

1. Go to **Google Takeout** at (https://takeout.google.com)
2. Click **Deselect all**.
3. Scroll down and turn on **Mail**.
4. Continue to the next step.
5. Choose to create the export.
6. Wait for Google to finish preparing your file.
7. Download the ZIP file Google sends you.

---

## How to Use This Project

1. Download your Google Takeout Mail ZIP.
2. Place the ZIP in the same location as your project code file.
3. Run the project code file.
4. It will read your Takeout Mail and show your results.
